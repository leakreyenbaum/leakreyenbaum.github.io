# Lea's Portfolio

A simple, elegant personal portfolio website with a coolwarm pastel color palette.

## Structure

- **index.html** - Main portfolio page with intro, projects, CV, thesis, and contact sections
- **css/** - Stylesheets
- **js/** - JavaScript files
- **public/** - Images and media assets

## Color Palette (Coolwarm Pastel)

- **Warm Coral**: `#FFA09B` - Primary accent color
- **Warm Coral Dark**: `#FF8B7D` - Hover states
- **Warm Peach**: `#FFD4C0` - Soft warm background
- **Cool Lavender**: `#E8D5F2` - Cool accent
- **Cool Blue**: `#B4D7FF` - Cool accent
- **Pastel Background**: `#FFF5F0` - Main page background
- **Text Main**: `#5A4A52` - Dark text
- **Text Muted**: `#8B7A8B` - Secondary text

## Features

- Bootstrap 5 responsive design
- Light/dark theme toggle
- Smooth animations and hover effects
- Mobile-friendly layout
- Font Awesome icons

## Setup

1. Place profile image at `public/profile.jpg`
2. Update the hero section with your name and title
3. Add CV and thesis links
4. Fill in the Projects, CV, and Thesis sections
5. Add contact information
